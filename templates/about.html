About

Anchoresi UI is build by Patrick Blaas and available on Github https://github.com/pblaas/anchoresi or https://hub.docker.com/r/pblaas/anchoresi/ to provide a simple interface for basic CRUD handeling on the Anchore platform.

It supports Adding, Deleting and Inspecting images with possible more features to come. It provides a alternative to the anchore-cli tool which is avaible from the Python 2.7 repo through pip, or the SAAS offering from Anchore.io although the functionality on Anchore.io is much more extensive then what Anchoresi is currently offering. I hope it will be of use to you.
